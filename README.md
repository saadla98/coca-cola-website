## ğŸ¬ Animations Breakdown

### **Framer Motion Animations**

#### **1. Scroll Animations**
```tsx
// Parallax effect
const { scrollYProgress } = useScroll()
const y = useTransform(scrollYProgress, [0, 1], [0, -100])

// Fade on scroll
<motion.div
  initial={{ opacity: 0, y: 50 }}
  whileInView={{ opacity: 1, y: 0 }}
  viewport={{ once: true }}
/>
```

#### **2. Hover Effects**
```tsx
<motion.div
  whileHover={{ 
    scale: 1.05, 
    y: -10,
    rotateY: 15 
  }}
  whileTap={{ scale: 0.95 }}
/>
```

#### **3. 3D Transforms**
```tsx
<motion.div
  style={{
    rotateY: currentIndex === index ? 0 : -35,
    scale: currentIndex === index ? 1.2 : 0.8,
    z: currentIndex === index ? 100 : 0,
  }}
/>
```
